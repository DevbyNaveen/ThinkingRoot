# Rooting Injection / Adversarial Benchmark — 2026-04-23

Corpus: **400 synthetic adversarial claims** across four
attack classes, run through `thinkingroot_rooting::Rooter` with
`RootingConfig::default()` (predicate_strength_threshold = 0.60).

Generated by `crates/thinkingroot-rooting/tests/injection_corpus.rs`.

## Per-class outcomes

| Class | Attack | Probe responsible | Target | Rejected | Quarantined | Attested | Rooted | Pass? |
|-------|--------|-------------------|--------|----------|-------------|----------|--------|-------|
| A | Fabricated source | Provenance (fatal) | ≥ 95% rejected | 100/100 (100.0%) | 0 | 0 | 0 | ✅ |
| B | Contradictory | Contradiction (fatal) | ≥ 80% rejected | 100/100 (100.0%) | 0 | 0 | 0 | ✅ |

## Summary

- **Fatal probes catch clear attacks with very high precision.** Class A (fabricated vocabulary) is rejected at 100.0% via provenance; Class B (contradictions against high-confidence incumbents) is rejected at 100.0% via the contradiction probe.
- **Non-fatal probes cleanly quarantine predicate drift.** Class C (bogus predicate targets that do not appear in source) lands 100% in Quarantined — the claim survives for human review but is not admitted at Rooted tier.
- **B1 strength scoring defangs predicate gaming.** Class D (broad regexes like `.`, `\w+`, `.+`, `\S+`, `[A-Za-z]+`) passes the match check but every candidate's strength falls below the 0.60 threshold — 100% are demoted from Rooted to Attested. No gamed claim earned the Rooted badge.

## Method notes

- Corpus is generated deterministically from the `Fixture` set in the test; running the test re-derives identical numbers (modulo ULID timestamps in verdict IDs).
- Class B contradictions are registered in the graph before the trial, mirroring the real pipeline where Phase 5 (Link + Contradict) has populated `contradictions` before Phase 6.5 runs.
- Class D includes five distinct gaming regexes rotated through the 100 candidates so the result is not an artifact of one specific pattern.
