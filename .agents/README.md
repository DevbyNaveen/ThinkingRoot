# Antigravity Workspace Config

This folder lets Gemini Antigravity load workspace-level agents, rules, skills, and workflows for ThinkingRoot.

Contents:
- `agents/` for workspace agent definitions
- `rules/` for project instructions
- `skills/` for reusable task knowledge
- `workflows/` for repeatable slash-command flows

The root `AGENTS.md` is the general instruction file for coding agents. `CLAUDE.md` remains the canonical detailed project memory for this repository.
